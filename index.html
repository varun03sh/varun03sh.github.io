Points System

  • Shortest Path: 200 RoutingCoins (with hover) / 800 RoutingCoins (without hover)
  • Valid Path: +400 RoutingCoins
  • Cell Points: -300 to +300 per cell
  • Grid Size Multipliers: 6x6 (1.0x), 7x7 (1.1x), 8x8 (1.2x), 9x9 (1.3x)
  • Penalty: -75 RoutingCoins per step beyond twice shortest path

Game Aim

Guide your player from the start point (top-left) to the end point (bottom-right) while collecting maximum virtual coins. Each cell has a point value that adds to your total score. Choose between taking the shortest path for a higher base reward or exploring high-point cells for potentially greater total rewards!
